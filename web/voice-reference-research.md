## RECOMMENDED EXTRACTION WORKFLOW

For YouTube clips:
1. Use `yt-dlp` to download audio: `yt-dlp -x --audio-format wav <URL>`
2. Open in Audacity or similar
3. Navigate to the recommended timestamp
4. Select a clean 10-30 second clip (no breaths at start/end)
5. Export as 16-bit 44kHz WAV
6. Verify with playback - listen for any room noise, mouth clicks, etc.

For LibriVox:
1. Download MP3 from librivox.org
2. Convert to WAV: `ffmpeg -i input.mp3 output.wav`
3. Select clean segment
4. Export

For VCTK:
1. Download from Edinburgh DataShare
2. Pick a speaker folder
3. Select any sentence recording
4. Already in ideal format for TTS
